1 шаг: установили git 
2 шаг: зашли в папку, которая не находится под версионным контролем git. Выполнили команду git init создав в данной папке локальный репозиторий.
3 шаг: создали файл "отчёт.txt" в папке "/home" и выполнив команду git add отчёт.txt добавили его под версионный контроль.
4 шаг: фиксируем проделанные шаги в отчете и создаем версию отчета в git с помощью команды.
5 шаг: создаем новую ветку в git с помощью команды git branch и вносим в неё изменения.
6 шаг: командой merge сливаем отчет в главную ветку.
7 шаг: регестрируемся на githab, создаем проект проект «homework IIT», заливаем содержимое из локального репозитория в проект на githab командой git push.
